# templates/

Starter templates and verbose output examples.

## Starter files

- `checkpoint.md` — Empty checkpoint with Knowledge State table and Next Task field. Copied to project root by `init-project.sh`.
- `implementation-plan.md` — Empty plan structure. Copied to project root or generated via plan mode.

## Output examples

Verbose examples moved here from `specs/` during prompt audit to keep specs concise:

- `synthesizer-synthesis.md` — Full example of synthesizer's synthesis narrative output
- `synthesizer-section-outline.md` — Full example of synthesizer's section outline output
- `triage-report.md` — Full example of triage agent's report output
- `triage-reading-plan.md` — Full example of triage agent's reading plan output

Agents reference these via their output format specs (e.g., `specs/synthesizer-output-format.md` points here for detailed examples).
