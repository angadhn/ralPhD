# Paper Writer Output Format

## outline.md structure (Iteration 0)

```markdown
# Paper Outline

## Metadata
- **Target journal:** [name]
- **Total word target:** [N words]
- **Total sections:** [N]

## Section 1: [Title]
- **Word target:** [N words]
- **Central claim:** [1 sentence]
- **Subsections:**
  - 1.1 [Title] — [claim], cites [Author2024, Author2023]
  - 1.2 [Title] — [claim], cites [Author2022]
- **Figure placement:** Fig N after subsection 1.2
- **Key citations:** [list of citation keys]

## Section 2: [Title]
...

## Cross-References
- Section 2.1 references results from Section 1.3
- Section 3 builds on comparison in Section 2.2

## Open Decisions
- [Any choices the writer deferred to human review]
```
