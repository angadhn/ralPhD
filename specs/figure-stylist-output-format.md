# Figure Stylist Output Format

## Round 1 — Full review (any issues)

```markdown
# Style Review — fig_NN_short_name

## Verdict: REVISE

### Must-Fix Issues
- [Issue 1 with specific fix instruction]
- [Issue 2 with specific fix instruction]

### Suggestions (optional)
- [Nice-to-have improvement]

### Round: 1
```

## Round 2 — Polish re-check (minor improvements only)

```markdown
# Style Review — fig_NN_short_name

## Verdict: APPROVED

### Minor Notes for Human
- [Any remaining cosmetic observations — informational only, not blocking]

### Round: 2
```

Round 2 always approves. If major issues persist, note them for the human — do not send back to the coder a third time.
